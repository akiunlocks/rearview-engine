field description
minutes 0-59 [, - * /]
hours 0-23 [, - * /]
days 1-31 [, - * ? / L W]
weekday select which days to run (otherwise it will run everyday)
month select which month to run (otherwise it will run everymonth)

The '*' character is used to specify all values. For example, "*" in the minute field means "every minute".

The '?' character is allowed for the days field. It is used to specify 'no specific value'. This is useful when you need to specify something in one of the weekday field.

The '-' character is used to specify ranges For example "10-12" in the hour field means "the hours 10, 11 and 12".

The ',' character is used to specify additional values. For example "MON,WED,FRI" in the day-of-week field means "the days Monday, Wednesday, and Friday".

The '/' character is used to specify increments. For example "0/15" in the seconds field means "the seconds 0, 15, 30, and 45".
